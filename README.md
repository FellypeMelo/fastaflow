# FastaFlow

Pipeline automatizado para processamento de arquivos FASTA.

## Instalação

```bash
pip install -r requirements.txt
```

## Uso

Execute: `python main.py`

## Funcionalidades

- Validação de arquivos FASTA
- Limpeza de sequências inválidas
- Cálculo de estatísticas básicas
- Integração com módulos anteriores
